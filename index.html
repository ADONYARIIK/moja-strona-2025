Co to jest DNS?

DNS (Domain Name System) to system, który tłumaczy adresy domenowe (np. www.google.com) na adresy IP, które rozumieją komputery (np. 142.250.74.36).

  • DNS działa jak książka telefoniczna dla internetu.
  • Każde urządzenie ma swój unikalny adres IP.
  • Użytkownik wpisuje nazwę domeny, a DNS odnajduje odpowiedni adres IP.

System DNS jest hierarchiczny: root (.) → top-level domains (.com, .pl) → domeny drugiego poziomu (np. google.com)

Przykład: Wpisujesz w przeglądarkę www.wikipedia.org:

  1. Komputer pyta lokalny serwer DNS o adres IP.
  2. Jeśli nie zna, pyta wyższy poziom DNS.
  3. DNS zwraca adres IP: 91.198.174.192
  4. Przeglądarka łączy się z tym adresem.
